# ImpactDash - NGO Tracking Dashboard 🇮🇳

ImpactDash is a modern, responsive, single-page application (SPA) designed to track verified Corporate Social Responsibility (CSR) funds, monitor Government NGO compliance (FCRA), and connect volunteers with local initiatives. 

This project consolidates fragmented NGO data into one unified interface, serving as a powerful tool for internship auditing and data visualization.

## 🚀 Features

* **Interactive Dashboard Overview:** Visualizes funding flows and focus area distributions using dynamic charts.
* **State & Fiscal Year Filtering:** Filters data dynamically across National, Maharashtra, Delhi NCR, Karnataka, and other regions.
* **NGO Directory & Compliance Tracking:** Displays active NGOs, their focus sectors, funding amounts, and FCRA verification status.
* **Dynamic Report Generation:** 
  * Export NGO Master Lists as `.csv` files.
  * Generate dynamic FCRA Compliance Audit `.txt` reports instantly.
* **Volunteers Hub:** Integrates directly with Google Forms to connect users with active volunteering drives based on their state.
* **Fully Mobile Responsive:** Features a professional desktop sidebar that seamlessly transitions into a thumb-friendly bottom navigation bar for mobile users.

## 🛠️ Tech Stack

This project was built to be lightweight, incredibly fast, and completely serverless.

* **Structure:** HTML5
* **Styling:** Tailwind CSS (via CDN)
* **Logic & Data Handling:** Vanilla JavaScript
* **Data Visualization:** Chart.js (via CDN)
* **Hosting:** Vercel

## 📂 File Structure

The entire application runs from a single, optimized file:
`index.html` - Contains the complete UI structure, Tailwind utility classes, and the embedded JavaScript logic for data filtering and chart rendering.

## 💻 How to Run Locally

Because this project does not rely on a backend database or complex frameworks, running it locally takes zero setup:
1. Download or clone this repository.
2. Double-click the `index.html` file to open it in any modern web browser (Chrome, Edge, Safari, Firefox).
3. The dashboard will instantly load and function entirely offline (Note: external links and Google Forms integration require internet access).

## 🌐 Live Deployment

This project is configured for seamless deployment on **Vercel**.
1. Push the `index.html` file to the root of a GitHub repository.
2. Import the repository into Vercel.
3. Vercel will automatically detect the static file and deploy it to a live, secure URL.
